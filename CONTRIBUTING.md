# How To Contribute

## Installation

* `git clone <repository-url>`
* `cd ember-headless-table`
* `pnpm install`

## Linting

* `pnpm lint`
* `pnpm lint:fix`

## Building the addon

* `cd ember-headless-table`
* `pnpm build`

## Running tests

* `cd test-app`
* `pnpm start` – starts the test app and tests are available at `/tests`
* `pnpm test:ember` – runs the ember tests for the current environment

## Running the docs app

* `cd docs-app`
* `pnpm start` – starts the test app and tests are available at `/tests`
